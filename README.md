# php-mysql-backend
Jednostavna web aplikacija koja uključuje mysql bazu podataka, php, backend i frontend. U ovom projektu uključen je kompletan CRUD sistem pomoću kojeg je napravljena struktura korisničkog interfejsa. Aplikacija omogućava dodavanje, prikaz, izmjenu i brisanje korisnika, dok se svi podaci čuvaju u bazi, što demonstrira osnovnu funkcionalnost dinamičnih web stranica.
